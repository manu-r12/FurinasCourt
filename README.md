A 2D narrative-driven game inspired by **Furina** from *Genshin Impact*, built using **Java** and **LibGDX**.

## About

FurinaCourt is an early-stage game project focusing on storytelling, choices, and court-style judgment mechanics in a 2D world inspired by Fontaine.

This repository currently contains the **base LibGDX setup** for desktop development.

## Tech Stack

- Java 17
- LibGDX
- Gradle`
- Desktop (Windows / macOS / Linux)
